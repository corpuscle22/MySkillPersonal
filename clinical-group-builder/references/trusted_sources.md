# Trusted Sources for Clinical Group Definitions

Cite authoritative sources when making clinical inclusion/exclusion decisions. Use the highest-tier source available.

## Tier 1 — Primary Authoritative Sources (Prefer These)

| Source | URL | Best For |
|--------|-----|----------|
| Medscape | emedicine.medscape.com | Test/drug indications, clinical workup |
| Merck Manual (Professional) | merckmanuals.com/professional | Disease-based diagnostic criteria |
| UpToDate | uptodate.com | Evidence-based clinical recommendations |
| PubMed | pubmed.ncbi.nlm.nih.gov | Primary literature, systematic reviews |
| Mayo Clinic | mayoclinic.org | Disease information, diagnostic approaches |
| Cleveland Clinic | clevelandclinic.org | Clinical practice, testing indications |

## Tier 2 — Institutional / Government Sources

| Source | URL | Best For |
|--------|-----|----------|
| NIH / NCBI | ncbi.nlm.nih.gov | Gene/enzyme info, GeneReviews, biochemistry |
| CDC | cdc.gov | Screening recommendations, public health |
| MedlinePlus | medlineplus.gov | Patient-facing test information |
| FDA | fda.gov | Drug approvals, safety, labeling |
| WHO | who.int | International guidelines, essential medicines |

## Avoid

- Wikipedia (use only for initial orientation, never cite)
- Health blogs, wellness sites
- Commercial lab marketing materials (use test *menus* for scope, not marketing copy)
- Non-peer-reviewed sources
- AI-generated medical content from unverified sources
