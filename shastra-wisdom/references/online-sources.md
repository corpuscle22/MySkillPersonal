## Primary Scripture Archives

| Source | URL | Best For | Notes |
|--------|-----|----------|-------|
| **Sacred Texts** | `sacred-texts.com/hin/` | All Hindu scriptures — Vedas, Upanishads, Gita, Puranas, Epics | Largest open archive; uses older (Griffith, Muir) translations — good for verse text, supplement with modern commentary |
| **Wisdom Library** | `wisdomlib.org` | Encyclopedic reference — Puranas, Arthashastra, Sanskrit terms, verse-by-verse | Excellent for Purana verses and Sanskrit term etymology |
| **Sanskrit Documents** | `sanskritdocuments.org` | Original Sanskrit texts, stotras, hymns | Best for getting original Sanskrit devanagari text |
| **GRETIL** | `gretil.sub.uni-goettingen.de` | Romanized Sanskrit e-texts of Hindu, Buddhist, Jain texts | Academic-grade Sanskrit e-texts |

## Search Strategy by Tier

### Tier 1 — Vedas & Upanishads
```
Primary: sacred-texts.com → sanskritdocuments.org → vedicheritage.gov.in
Commentary: gitasupersite.iitk.ac.in (for Upanishads with commentary) → saksi.org
Verification: Cross-check verse number with wisdomlib.org
```

### Tier 2 — Gita, Epics, Brahma Sutras
```
Primary: gitasupersite.iitk.ac.in (Gita) → valmikiramayan.net (Ramayana) → sacred-texts.com/hin/mbs/ (Mahabharata)
Commentary: Multiple acharya views on gitasupersite
Verification: Cross-check with sanskritdocuments.org for original Sanskrit
```

### Tier 3 — Puranas & Dharma Shastras
```
Primary: wisdomlib.org → srimadbhagavatam.org → sacred-texts.com
Commentary: vedabase.io (Vaishnava) → archive.org (classical commentaries)
Verification: Cross-check with a second source from the list
```

### Tier 4 — Specialized Texts
```
Primary: sanskritdocuments.org → sacred-texts.com → wisdomlib.org
Commentary: Relevant tradition site (dvaita.org, ramanuja.org, kamakoti.org)
Verification: Cross-check with archive.org scanned originals
```
