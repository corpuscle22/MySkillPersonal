# Trusted Medical Reference Sources

Reference sources to cite when providing clinical indications. Use these authoritative sources and include URLs.

## Tier 1: Primary Medical References (Prefer These)

| Source | Domain | Description |
|--------|--------|-------------|
| **Medscape** | medscape.com | Professional medical reference with disease/drug information |
| **Merck Manual** | merckmanuals.com | Comprehensive medical textbook (Professional & Consumer editions) |
| **UpToDate** | uptodate.com | Evidence-based clinical decision support |
| **PubMed/NCBI** | pubmed.ncbi.nlm.nih.gov | Peer-reviewed medical literature database |
| **Mayo Clinic** | mayoclinic.org | Patient and professional medical information |
| **Cleveland Clinic** | clevelandclinic.org | Healthcare information and guidelines |

## Tier 2: Government & Institutional Sources

| Source | Domain | Description |
|--------|--------|-------------|
| **NIH** | nih.gov | National Institutes of Health resources |
| **CDC** | cdc.gov | Disease prevention and health information |
| **MedlinePlus** | medlineplus.gov | NIH consumer health information |
| **FDA** | fda.gov | Drug and device labeling information |
| **WHO** | who.int | Global health guidelines |

## Tier 3: Specialty Sources

| Source | Domain | Description |
|--------|--------|-------------|
| **ARUP Consult** | arupconsult.com | Lab test selection and interpretation |
| **Lab Tests Online** | labtestsonline.org | Patient-focused lab test information |
| **ACR (Radiology)** | acr.org | Imaging appropriateness criteria |
| **AHA (Cardiology)** | heart.org | Cardiovascular guidelines |

## Sources to AVOID

Do NOT cite these as primary references:
- Wikipedia (use only for background)
- Generic health blogs
- Non-peer-reviewed sources
- Commercial lab marketing materials
- Social media or forums

## Citation Format

When citing clinical indications, use this format:

```
- [Disease/Condition] — [Source Name](URL)
```

**Example:**
```
Clinical Indications:
- Rheumatoid Arthritis — [Medscape](https://emedicine.medscape.com/article/331715-workup)
- Systemic Lupus Erythematosus — [Merck Manual](https://www.merckmanuals.com/professional/musculoskeletal-and-connective-tissue-disorders/systemic-lupus-erythematosus)
```
