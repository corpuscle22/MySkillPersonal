# Clinical Indication Patterns by Test Category

Reference for common clinical indications organized by laboratory test category.

## Chemistry Tests (CHEM)

### Glucose Tests
- Diabetes mellitus (Type 1 and Type 2)
- Prediabetes / Impaired glucose tolerance
- Hypoglycemia
- Gestational diabetes
- Diabetic ketoacidosis (DKA)
- Hyperosmolar hyperglycemic state (HHS)
- Insulinoma
- Metabolic syndrome

### Hemoglobin A1c
- Diabetes mellitus monitoring and diagnosis
- Prediabetes screening
- Glycemic control assessment

### Lipid Panel (Cholesterol, Triglycerides, HDL, LDL)
- Dyslipidemia
- Hypercholesterolemia
- Hypertriglyceridemia
- Cardiovascular disease risk assessment
- Atherosclerosis

### Electrolytes (Sodium, Potassium, Chloride, CO2)
- Dehydration / Fluid imbalance
- Hyponatremia / Hypernatremia
- Hypokalemia / Hyperkalemia
- Metabolic acidosis / alkalosis
- Renal disease
- Adrenal insufficiency

### Kidney Function (BUN, Creatinine, eGFR)
- Acute kidney injury
- Chronic kidney disease
- Nephrotoxicity monitoring
- Prerenal azotemia
- Glomerulonephritis

### Liver Function (AST, ALT, ALP, Bilirubin, Albumin)
- Hepatitis (viral, alcoholic, drug-induced)
- Cirrhosis
- Cholestasis / Biliary obstruction
- Liver metastases
- Hepatocellular carcinoma

### Thyroid Function (TSH, T4, T3)
- Hypothyroidism
- Hyperthyroidism
- Graves disease
- Hashimoto thyroiditis
- Thyroid nodules
- Thyroid cancer


## Hematology Tests (HEM/BC)

### Complete Blood Count (CBC)
- Anemia (iron deficiency, B12/folate deficiency, aplastic)
- Polycythemia
- Leukemia / Lymphoma
- Infection
- Thrombocytopenia / Thrombocytosis
- Myelodysplastic syndrome

### Hemoglobin / Hematocrit
- Anemia
- Blood loss
- Polycythemia vera
- Dehydration

### White Blood Cell Count
- Infection (bacterial, viral)
- Leukemia
- Immunodeficiency
- Autoimmune disorders

### Platelet Count
- Thrombocytopenia
- Immune thrombocytopenic purpura (ITP)
- Thrombotic thrombocytopenic purpura (TTP)
- Disseminated intravascular coagulation (DIC)


## Coagulation Tests (COAG)

### PT/INR
- Warfarin monitoring
- Liver disease
- Vitamin K deficiency
- Coagulation factor deficiency

### PTT/aPTT
- Heparin monitoring
- Hemophilia
- Lupus anticoagulant
- von Willebrand disease

### D-dimer
- Deep vein thrombosis (DVT)
- Pulmonary embolism (PE)
- Disseminated intravascular coagulation (DIC)


## Immunology/Serology (SERO)

### Antinuclear Antibody (ANA)
- Systemic lupus erythematosus (SLE)
- Sjogren syndrome
- Scleroderma
- Mixed connective tissue disease

### Rheumatoid Factor
- Rheumatoid arthritis
- Sjogren syndrome
- Cryoglobulinemia


## Cardiac Markers

### Troponin
- Acute myocardial infarction
- Acute coronary syndrome
- Myocarditis
- Pulmonary embolism (severe)

### BNP / NT-proBNP
- Heart failure
- Cardiac dysfunction
- Pulmonary hypertension


## Tumor Markers

### PSA (Prostate-Specific Antigen)
- Prostate cancer screening
- Prostate cancer monitoring
- Benign prostatic hyperplasia

### CA-125
- Ovarian cancer
- Ovarian cancer monitoring
- Endometriosis

### CEA (Carcinoembryonic Antigen)
- Colorectal cancer monitoring
- Gastrointestinal malignancies
- Lung cancer


## Urinalysis (UA)

- Urinary tract infection (UTI)
- Kidney disease
- Diabetes mellitus
- Nephrolithiasis
- Glomerulonephritis
- Proteinuria
